Ridiculously Responsive Social Sharing Buttons

Love them or hate them (or even call them unnecessary), social sharing buttons appear to be with us for a while. It seemed like we were constantly making custom buttons for every single project, so we decided to create a super flexible system that would work in any container.

While these particular buttons original intent was for long form post sharing, they can work for any use. RRSSB is build with SASS, and svgs, so its retina ready out of the box. Add or remove icons as you see fit, the rest will fill in the space. Joshua, should we show demo of different containers?

Grab the code on Github. Scale this page for demo. Currently tested between 200px and 3400px.
